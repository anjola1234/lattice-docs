---
title: "Search Nodes"
description: "Search for nodes by alias"
---

Searches for nodes whose alias matches the given query string.

## Request
GET /api/nodes/search

### Query Parameters

<ParamField query="q" type="string" required>
  The search term to match against node aliases. Case-insensitive, partial matches supported.
</ParamField>

## Example Request

```bash
curl "https://lattice-blue.vercel.app/api/nodes/search?q=bitcoin"
```

## Response Fields

| Field | Type | Description |
|---|---|---|
| `results` | array | List of matching nodes |
| `total` | number | Total number of matches found |
| `query` | string | The search term that was used |

Each object in `results` follows the same node structure described in [Get Node](/nodes/get-node).
