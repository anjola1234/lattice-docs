---
title: "Top Nodes"
description: "Retrieve the highest-ranked nodes by channel count"
---

Returns the top nodes ranked by number of channels, in descending order.

## Request
GET /api/nodes/top

### Query Parameters

<ParamField query="limit" type="number" default="10">
  The maximum number of nodes to return.
</ParamField>

## Example Request

```bash
curl "https://lattice-blue.vercel.app/api/nodes/top?limit=3"
```

## Response Fields

| Field | Type | Description |
|---|---|---|
| `nodes` | array | List of top nodes, ordered by channel count descending |
| `limit` | number | The limit that was applied to this request |
| `total` | number | Number of nodes returned |

Each object in `nodes` follows the same node structure described in [Get Node](/nodes/get-node).
