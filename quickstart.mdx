This guide walks through making your first request to the Lattice API.

## Get network statistics

The simplest way to confirm the API is working is to fetch overall graph statistics.

```bash
curl https://lattice-blue.vercel.app/api/graph/stats
```

This returns a summary of the entire Lightning Network graph snapshot:

## Look up a specific node

You can fetch details for a single node using its public key:

```bash
curl https://lattice-blue.vercel.app/api/nodes/02000200020002000200020002000200020002000200020002000200020002abcd
```

This returns the node's alias, capacity, channel count, centrality score, and a list of its directly connected neighbors.
